# clean-reset-css

A minimal CSS reset.

No framework.
No design decisions.
No hidden behavior.

Just a clean, predictable starting point.

---

## Why?

Most resets try to solve everything.

They normalize browsers, define typography, override elements,
and often introduce more complexity than they remove.

This one doesn’t.

> Reset ≠ Design system

---

## What it does

- Sets consistent `box-sizing`
- Removes default margins
- Improves media handling
- Normalizes form elements
- Prevents text overflow issues

That’s it.

---

## What it does NOT do

- No typography system
- No colors
- No layout rules
- No accessibility hacks
- No framework-style overrides

Everything else is your job — intentionally.
